Service

```service
apiVersion: v1
kind: Service
metadata:
  name: [app-label]
  namespace: [namespace]
spec:
  type: ClusterIP
  selector:
    app: [app-label]
  ports:
    - protocol: TCP
      port: [port]
      targetPort: [targetport]
```

```service
        matchLabels:
          app: [app-label]
    ports:
    - port: [port]
      protocol: TCP
  podSelector: {}
```
